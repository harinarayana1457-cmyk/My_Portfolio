
# My Digital Portfolio

A modern, responsive personal portfolio designed to showcase my journey as a developer, highlighting my projects, technical expertise, and professional achievements.


## Tech Stack

Core: React.js, Tailwind CSS

Deployment: Netlify / Vercel

Backend/Tools: Firebase, Google Cloud Platform

Design: Framer Motion, Lucide Icons, Superbase


## Features

- Interactive Project Gallery: Showcasing work like Project Sentinel and the FFCS Time Table Architect.
- Technical Certifications: Integrated display of badges, including the Google Kubernetes Skill Badge.
- Real-time Responsiveness: Optimized for all devices using modern CSS grid and flexbox logic.
- Automated Workflow: Built with a focus on clean code and rapid deployment.
